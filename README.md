## 🏗️ Technical Portfolio Ecosystem

Below is a deep dive into the engineering projects developed to push the boundaries of digital construction.

### 🧠 1. BIM-Lawyer: Enterprise Compliance Engine
An automated auditing system for Revit models designed to ensure strict adherence to local laws and ABNT/ISO standards. It uses a hybrid RAG (Retrieval-Augmented Generation) architecture to normalize multi-unit parameters and cross-reference them with legal databases.

*   **Core Architecture**: Modular auditing kernel with specialized unit normalization.
*   **Technologies**: 
    - **Languages**: Python 3.12, C# (.NET 8)
    - **Frameworks**: FastAPI, LangChain, WPF (Revit UI)
    - **AI/ML**: OpenAI/Gemini Embeddings, ChromaDB (Vector Store)
    - **BIM**: Autodesk Revit API 2024
    - **Infrastructure**: Docker, Docker Compose, Enterprise Logging

### 🍃 2. EcoBIM-Logic: Sustainability Core
A high-performance calculation engine for Whole Life Carbon Assessment (WLCA) compliant with EN 15978 and EN 15804. It integrates directly with BIM environments to provide real-time carbon footprints from the earliest design stages (Modules A1-D).

*   **Core Architecture**: Multi-phase carbon calculator with material recommendation engine.
*   **Technologies**: 
    - **Languages**: Python 3.12, C# (.NET 8)
    - **Frameworks**: FastAPI, Uvicorn, Pydantic v2
    - **Math/Analysis**: NumPy, SciPy (Mass-compensation algorithms)
    - **AI/ML**: Scikit-learn (KNN for material classification)
    - **BIM**: Autodesk Revit API 2025
    - **Reporting**: ReportLab, Matplotlib (Automated PDF Generation)

### 📊 3. Production-Dashboard-AI: Agentic Analytics
A real-time production monitoring dashboard for heavy machinery. It leverages a natural language "Agentic AI Workflow" where commands are parsed into structured data for dynamic visual updates.

*   **Core Architecture**: Event-driven agentic workflow with Firebase synchronization.
*   **Technologies**: 
    - **Frontend**: Vanilla JavaScript (ES6+), Tailwind CSS v4, Vite
    - **Backend/DB**: Firebase Firestore, Firebase Auth
    - **Intelligence**: Google Gemini 2.5 API
    - **Visualization**: Chart.js

### 🌐 4. Nexus-Twin: Structural Health Digital Twin
A framework for real-time Structural Health Monitoring (SHM). It synchronizes IoT sensor data with 3D models to forecast structural failures and detect anomalies using machine learning.

*   **Core Architecture**: Dual-layer anomaly detection (Statistical + Regressive forecasting).
*   **Technologies**: 
    - **Languages**: Python 3.12, JavaScript
    - **AI/ML**: Scikit-learn (RandomForest Regressor, IsolationForest)
    - **Database**: SQLite (aiosqlite)
    - **Visualization**: Three.js (Digital Twin rendering)
    - **Reporting**: ReportLab

### 👁️ 5. AuraVision: Pathology Detection CV
Computer vision system specialized in the identification of construction pathologies like cracks and efflorescence. It uses advanced image filtering to isolate structural issues and estimate volume/severity.

*   **Core Architecture**: Hessian ridge detection pipeline with volumetric estimation engine.
*   **Technologies**: 
    - **Languages**: Python 3.12
    - **Libraries**: OpenCV (cv2), SciPy (Frangi filters), NumPy, Matplotlib
    - **API**: FastAPI

### 🔗 6. BIM-to-Graph: Spatial Topology Engine
An ETL pipeline that transforms physical IFC models into mathematical graph structures (Nodes & Edges) for advanced topological and connectivity analysis.

*   **Core Architecture**: Connectivity-first graph extraction from IFC spatial hierarchies.
*   **Technologies**: 
    - **Engine**: IfcOpenShell, NetworkX
    - **Analysis**: NumPy (Dijkstra's shortest path)
    - **Visualization**: Pyvis, Matplotlib

### 🕶️ 7. SiteSense-AR: WebXR Construction Monitoring
A lightweight Augmented Reality solution for verifying construction progress on-site without native app dependencies.

*   **Core Architecture**: Mobile-first WebXR scaffold for BIM overlay.
*   **Technologies**: 
    - **Core**: Three.js, WebXR Device API
    - **Build**: Vite, JavaScript (ES6)

### 📦 8. OpenIFC-DataWrangler: IFC Data Engineering
A robust data engineering tool for extracting and cleaning metadata from complex IFC files to facilitate automated quantity takeoffs.

*   **Core Architecture**: ETL workflow with relational normalization of IFC schemas.
*   **Technologies**: 
    - **Core**: IfcOpenShell, Pandas, NumPy
    - **Database**: SQLite3, SQLAlchemy, Pydantic v2

### 🤖 9. AECAgent-RAG: Technical Standards AI
A specialized RAG-based AI agent designed to parse complex, multi-column AEC normative documents and answer technical questions with high precision.

*   **Core Architecture**: Normative article boundary detection kernel.
*   **Technologies**: 
    - **Frameworks**: LangChain, Google Gemini API
    - **Parsing**: PyPDF2, Unstructured, ChromaDB
